# 三. 继承与反射

这一章主要介绍面向对象程序设计的另外一个基本概念,那就是**继承 (inheritance)**.利用继承,可以基于已存在的类构造一个新类.继承已存在的类就是复用这些类的方法和域.这是 Java 程序设计中的一项核心技术.

本章还介绍了**反射 (reflection)**的概念.反射式值在程序运行期间发现更多的类及其属性的能力.这是一个功能强大的特性,使用起来比较复杂.
